# Завдання 3. Алгоритм Дейкстри з використанням бінарної купи

## Опис завдання

Метою завдання є програмна реалізація алгоритму Дейкстри
для знаходження найкоротших шляхів у зваженому графі
з використанням бінарної купи (піраміди).

Алгоритм повинен:
- працювати з графом;
- використовувати піраміду для оптимізації вибору вершини;
- знаходити найкоротші відстані від початкової вершини до всіх інших.
