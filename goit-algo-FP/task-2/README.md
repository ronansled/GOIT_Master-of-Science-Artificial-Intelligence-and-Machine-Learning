# Завдання 2. Рекурсія. Фрактал “Дерево Піфагора”

## Опис завдання

Метою завдання є використання рекурсії для побудови фракталу
“Дерево Піфагора” з можливістю керування глибиною рекурсії користувачем.

Програма повинна:
- використовувати рекурсивний алгоритм;
- візуалізувати фрактал;
- дозволяти користувачу задавати рівень рекурсії.

---

## Реалізація

Фрактал реалізовано за допомогою бібліотеки `turtle`.

Кожна гілка:
- малюється як відрізок;
- породжує дві нові гілки;
- зменшується у розмірі;
- зникає після повернення з рекурсії (backtracking).

Рекурсія припиняється, коли рівень дорівнює 0.
