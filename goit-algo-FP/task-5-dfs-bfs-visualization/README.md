# Завдання 5. Візуалізація обходу бінарного дерева (DFS / BFS)

## Опис завдання

Метою завдання є візуалізація обходу бінарного дерева:
- у ширину (BFS);
- у глибину (DFS).

Для реалізації необхідно:
- використати код побудови дерева з попереднього завдання;
- застосувати стек та чергу;
- не використовувати рекурсію;
- змінювати колір вузлів у порядку обходу;
- використовувати HEX-кольори (#RRGGBB).

---

## Реалізація обходів

### BFS (обхід у ширину)
- реалізовано за допомогою `deque`
- використовується черга (FIFO)

### DFS (обхід у глибину)
- реалізовано за допомогою списку
- використовується стек (LIFO)

---

## Кольорова індикація

- перші вузли — темніші
- останні — світліші
- колір залежить від порядку відвідування
