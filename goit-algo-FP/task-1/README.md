# Завдання 1. Однозв'язний список. Сортування

## Опис завдання

Метою цього завдання є реалізація базових операцій над однозв'язним списком,
а також застосування алгоритмів сортування та злиття списків.

---

## Реалізовані вимоги

### 1. Реверсування однозв'язного списку
Реалізовано функцію `reverse`, яка змінює напрямок посилань між вузлами
без створення нового списку.

**Складність:**
- Часова: O(n)
- Просторова: O(1)

---

### 2. Сортування однозв'язного списку
Використано **сортування злиттям (merge sort)**, яке є оптимальним
для однозв'язних списків.

**Причини вибору:**
- Не потребує випадкового доступу
- Гарантована складність O(n log n)

---

### 3. Об'єднання двох відсортованих списків
Реалізована функція `merge_two_sorted_lists`, яка:
- приймає два відсортовані однозв'язні списки
- повертає новий відсортований список

**Складність:**
- Часова: O(n + m)
- Просторова: O(1) (без урахування нового списку)

---
