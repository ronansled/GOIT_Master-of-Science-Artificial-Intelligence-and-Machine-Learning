# AVL Tree — Sum of All Values

## Опис задачі
Реалізувати алгоритм, який знаходить суму всіх значень
у двійковому дереві пошуку або AVL-дереві.

## Вибране рішення
Використано AVL-дерево (самобалансуюче BST).

Для обчислення суми використовується
рекурсивний обхід дерева (DFS).

Формула:
sum(node) = node.value + sum(left) + sum(right)

Порядок обходу не має значення, оскільки
операція додавання комутативна.

## Складність
- Часова: **O(n)** — необхідно відвідати кожен вузол
- Просторова: **O(h)** — глибина рекурсії (h — висота дерева)
