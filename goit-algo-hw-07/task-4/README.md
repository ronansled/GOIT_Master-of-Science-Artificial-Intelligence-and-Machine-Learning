# Hierarchical Comment System

## Опис задачі
Реалізувати структуру даних для системи коментарів,
де кожен коментар може мати довільну кількість відповідей,
які також можуть мати свої відповіді.

Структура формує ієрархічне дерево.

## Основні вимоги
- Коментар зберігає:
  - текст
  - автора
  - список відповідей
- Можливість:
  - додавати відповіді
  - логічно видаляти коментар
  - рекурсивно відображати всю структуру

## Реалізація
Використано клас `Comment`.

Особливості:
- Видалення є логічним (`is_deleted = True`)
- Відповіді зберігаються навіть після видалення коментаря
- Вивід використовує рекурсію та відступи (4 пробіли)
