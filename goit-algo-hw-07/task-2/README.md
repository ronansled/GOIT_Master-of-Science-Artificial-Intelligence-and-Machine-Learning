# AVL Tree — Finding Minimum Value

## Опис задачі
Реалізувати алгоритм, який знаходить найменше значення
у двійковому дереві пошуку або AVL-дереві.

## Вибране рішення
Використано AVL-дерево — самобалансуюче BST.

Ключова властивість:
> У будь-якому BST або AVL-дереві найменше значення
> завжди знаходиться у крайньому лівому вузлі.

Алгоритм:
- почати з кореня;
- рухатися вліво, доки існує лівий нащадок.

## Складність
- **O(log n)** для AVL-дерева
- **O(h)** у загальному випадку BST
