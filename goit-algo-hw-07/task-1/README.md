# AVL Tree — Finding Maximum Value

## Опис задачі
Реалізувати алгоритм, який знаходить найбільше значення
у двійковому дереві пошуку або AVL-дереві.

## Вибране рішення
Використано AVL-дерево, яке є самобалансуючим BST.

Ключова властивість:
> У будь-якому BST або AVL-дереві найбільше значення
> завжди знаходиться у крайньому правому вузлі.

Тому алгоритм пошуку максимуму:
- почати з кореня;
- рухатися вправо, доки існує правий нащадок.

Складність:
- **O(log n)** для AVL-дерева.
